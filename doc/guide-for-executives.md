# 助成金BOTのご案内

## これは何？

毎月1回、HAMAYOUリゾートが使えそうな助成金・補助金の情報を自動で集めて、ChatWorkに届けてくれるBOTです。

今までは「たまたま知った」「誰かが教えてくれた」で助成金の情報を得ていましたが、このBOTが毎月決まったタイミングで届けてくれるので、「知らなかったから使えなかった」がなくなります。

---

## いつ届くの？

**毎月 第1木曜日の朝**に、ChatWorkの助成金チャンネルに届きます。
その日の役員会議の冒頭で、届いた内容を見ながら話し合う流れです。

---

## どんな形で届くの？

候補が複数ある月は、最初に**一覧メッセージ**が届きます。
「今月は3件あります。急ぎのものから順に並べています」のような案内です。

その後に**1件ずつの詳細メッセージ**が届きます。それぞれに以下が書いてあります。

- こんな使い方ができそう（例: フロントの省人化、ボイラーの更新 など）
- 締切はいつか
- 準備にどのくらい時間がかかりそうか
- いくらくらい補助が出るか
- メリットと注意点
- この場で何を決めればよいか

候補が0件の月は「今月は該当なし」のメッセージが1通届くだけです。

---

## 会議では何をすればいいの？

届いたメッセージを上から順に見ていき、それぞれについて以下のどれかを決めてください。

- **採用** → 担当と期限を決めて、申請準備に入る
- **条件付き採用** → もう少し調べてから最終判断する
- **持越し** → 来月以降にあらためて検討する
- **見送り** → 今回は使わない

決まったら、そのメッセージを**ChatWorkのタスクにする**だけでOKです。
担当者と期限を入れれば、記録が残ります。

---

## 採用した後はどうなるの？

実際の申請作業（書類作成・提出など）は、制度に応じて**専門家（社労士・行政書士など）に依頼**します。BOTが申請書を書くわけではありません。

BOTの役割は「見つけて届けるところまで」です。
採用後は担当者が専門家と相談しながら進めます。

---

## どのくらいの調査をしているの？

初回（2026年4月）の実績です。

### 巡回した情報源

省庁・県・町の公式サイトを合計**25サイト**登録しています（毎月必ず見るもの12サイト、テーマ別8サイト、補助的5サイト）。今回は主要12サイトを中心に巡回し、**28件の制度・情報**をピックアップしました。

| 情報源 | 見つけた件数 |
|--------|------------|
| ミラサポplus（中小企業庁） | 6件 |
| 厚生労働省 雇用関係助成金 | 4件 |
| 観光庁 公募情報 | 3件 |
| 富士河口湖町 中小企業支援 | 3件 |
| 山梨労働局 | 3件 |
| 中小企業省力化投資補助金 | 2件 |
| 山梨県 省エネ・賃上げ補助 | 2件 |
| その他5サイト | 各1件 |

### 調査の進め方

4方向の調査を同時並行で実施しています。

| 調査方向 | 調査量 |
|---------|--------|
| 全国系の主要補助金 | 約350行分 |
| 山梨県・富士河口湖町の地域補助 | 約450行分 |
| テーマ別（省エネ・観光・外国人材など） | 約560行分 |
| 固定費削減に特化（ソーラー・省人化など） | 約580行分 |
| **合計** | **約1,900行分の調査メモ** |

### 絞り込み

この約1,900行の調査メモから、重複を除き、HAMAYOUに当てはまらない制度を除外する**罠チェック**（規模条件・業種ミスマッチなど9項目）を経て、最終的に以下に整理しています。

| 区分 | 件数 | 意味 |
|------|------|------|
| 採否候補 | 6件 | 会議で判断する候補 |
| 監視継続 | 7件 + 制度ウォッチ1件 | 今月は対象外だが来月以降に動きがありそう |
| 除外（近接候補） | 3件 | 一見使えそうだがHAMAYOUには合わない |

人手でやると数日かかるような調査を、BOTが毎月自動で行います。

---

## 何ができて、何ができないのか

| できること | できないこと |
|-----------|-------------|
| 毎月の助成金情報を自動で収集 | 申請書の作成 |
| HAMAYOUに合いそうなものだけ選別 | 申請の代行 |
| 会議で判断しやすい形に整理して通知 | 専門家の代わり |
| 採用した案件の進捗・効果の記録管理 | 採択の保証 |

---

## まとめ

- **毎月第1木曜の朝**にChatWorkに届く
- **会議で読み上げて、採用/見送りを決める**だけ
- **採用したら担当と期限を決めてタスク化**する
- 申請の実務は**専門家に依頼**する
- 皆さんにお願いしたいのは**会議で判断すること**だけです

# かかった費用と工数

> **いつ書く？** 年に1回、振り返りのタイミングで（または入金後にまとめて）
> **何を書く？** 実際にいくらかかったか、どのくらい手間がかかったかを記録します
> **なぜ必要？** 「来年も同じ制度に申請すべきか」「外注した方がよかったか」を判断するための材料になります
> **所要時間:** 30分くらい（まとめて振り返るとき）

---

## お金の記録

申請から入金までの間に数字がわかり次第、埋めてください。
最終的に「入金額 − 外注費 − 社内工数」で本当に得したかがわかります。

| 項目 | 金額 | いつわかるか |
|------|------|-------------|
| 申請額 | | 申請時 |
| 交付決定額 | | 交付決定時（申請額と違うことがある） |
| 確定額 | | 実績報告後（減額されることがある） |
| 入金額 | | 入金時（最終的に振り込まれた額） |
| 外注費（社労士・コンサル等） | | 支払い時 |

## 手間の記録

ざっくりでOKです。「1時間」「半日」「丸1日」くらいの粒度で。
どのフェーズに手間がかかったかがわかると、次回の判断に役立ちます。

| フェーズ | かかった時間 | 誰が | メモ |
|----------|-------------|------|------|
| 候補選定・検討 | | | 例: 会議で30分 |
| 申請書類の作成 | | | 例: 和田が丸1日 |
| 業者との調整 | | | 例: 見積もり3社で半日 |
| 実績報告 | | | 例: 写真撮影+報告書で1日 |
| 事後報告（毎年） | | | 例: 毎年半日 |

## 設計方針（確定済み）

1. 個別通知は **「制度説明文」ではなく「会議用判断カード」**
2. 冒頭3行は **想定使途 / 締切・準備余裕 / 金額感**
3. 適合度の単一スコア（◎○△×）はやめる → **2軸（事業適合×準備余裕）+ 仮判断ラベル**
4. 本文は **採る理由 / 実行条件 / 主な負担・制約** の3部構成
5. 「リードタイム」→「準備余裕」に改名し、**間に合わせる条件** を併記
6. ChatGPT相談プロンプトは本文埋め込みをやめ、**別メッセージリンク** に
7. 会議アクションは **4択**（採用 / 条件付き採用 / 持越し / 見送り）+ 担当 + 次期限
8. サマリは **議事進行表**（推奨順・仮判断・準備余裕・個別リンク）
9. 投稿順は **個別 → サマリ**（サマリに個別へのリンクを貼るため）
10. BOTは **専用ChatWorkアカウント** または `self_unread` を使う（自分宛て未読対策）

---

## 仮判断ラベル定義

| ラベル | 意味 | サマリ区分 |
|---|---|---|
| 採用候補 | 適合・準備余裕ともに高い、速決推奨 | C: 速決候補 |
| 条件付き採用 | 適合は高いが準備余裕に課題、条件付き | A: 本日判断したい |
| 持越し候補 | 情報追加が必要 | B: 情報追加後に判断 |
| 見送り候補 | 適合度が低い／規模条件で対象外 | 最下段 |
| 要確認 | 判定困難（×の代わり） | 適宜 |

## 個別通知テンプレ

```
[info]
[title]① 中小企業省力化投資補助金（一般型）｜光風閣くわるび[/title]
■ 想定使途: フロント業務の省人化
■ 締切/準備余裕: 2026-05-15（あと34日） / 低
■ 金額感: 補助率 1/2・上限 750万円
■ 今日の仮判断: 条件付き採用

【一言でいうと】
金額感は大きい。今回出すなら、今週中に見積・担当・外部着手枠を固める必要あり。

【採る理由】
- 光風閣くわるびの省人化に直結
- 団体受入オペレーション改善と相性が良い

【実行条件】
- preflight の赤項目がないこと
- 対象施設を光風閣くわるびに固定
- 今週中に必要資料の取得可否を確認

【主な負担・制約】
- 同一内容の他制度との重複確認が必要
- 事後報告の負担あり
- 先払い負担が発生する可能性あり

【会議で決めること】
□ 採用
□ 条件付き採用
□ 持越し
□ 見送り

担当候補: 和田 / 管理部
次期限: YYYY-MM-DD
持越し時の相談テンプレ: [相談テンプレメッセージリンク]
[/info]
```

## 運用ルール

- 投稿順: 個別1 → 個別2 → 個別3 → サマリ
- 会議後: 個別通知への返信で **【会議決定】** を定型投稿
- 採用・条件付き採用: その場で **ChatWorkタスク化**（担当・期限）
- 投稿は **BOT専用アカウント** または `self_unread`

# ChatWork通知フォーマット（正本）

このファイルは、月次BOTがChatWorkに投稿する通知メッセージのフォーマット仕様の正本です。

---

## ステータス

**確定**（2026-04-11 フォーマット改善・動作確認済み）。

---

## 設計方針（確定済み）

1. 個別通知は **「制度説明文」ではなく「会議用判断カード」**
2. 冒頭は **使い方の例 / 締切 / 準備時間 / 金額感** を4行で示す
3. 断定口調を避け、**提案・可能性のニュアンス** で書く（「など」「〜の場合」「可能性がある」）
4. 本文は **メリット / 申請に必要なこと / 申請〜運用の負担** の3部構成
5. 各選択肢には **選んだ先にどうなるか** を併記する
6. 投稿順は **2件以上: サマリ → 個別**、**1件: 個別のみ**
7. サマリの冒頭に **何件あってどう読めばよいか** を自然な文章で説明
8. 会議後の記録は個別メッセージへの **返信** で残す
9. 採用・条件付き採用は **ChatWorkタスク化**（担当・期限）
10. 投稿は **BOT専用ChatWorkアカウント** または `self_unread` で投稿

---

## 投稿ルール

| 候補件数 | 投稿内容 |
|---|---|
| 0件 | 「今月は0件」メッセージ1通のみ |
| 1件 | 個別メッセージ1通のみ（サマリなし） |
| 2件以上 | サマリ1通 → 個別メッセージN通 |

## 個別通知テンプレ

```
[info]
[title]① IT導入補助金（通常枠）[/title]
■ 例えばこんな使い方: 光風閣くわるびの予約管理システム刷新 など
■ 締切: 2026-05-30
■ 準備にかかる時間の目安: ふつう（IT導入支援事業者の選定が必要、2週間程度）
■ 金額感: 補助率1/2、上限450万円

【一言でいうと】
上記のような使い方をする場合、予約管理の手作業を減らせる。
支援事業者さえ決まれば申請は比較的スムーズ。

【この制度を使うメリット】
- 光風閣くわるびのフロント業務効率化につながる可能性がある
- 団体予約の管理工数を大幅に削減できる見込み

【申請するために必要なこと】
- IT導入支援事業者の選定・契約
- 導入するITツールの事前確認

【申請〜運用で発生する負担】
- 支援事業者との調整に工数がかかる
- 導入後、効果の実績報告を提出する必要がある

【この場で決めていただきたいこと】
以下のうち、どれにしますか？
□ 採用 → 担当と期限を決めて申請準備に入る
□ 条件付き採用 → 追加情報を確認してから最終判断
□ 持越し → 来月以降にあらためて検討
□ 見送り → 今回は対象外

→ 採用または条件付きの場合:
　担当: （この場で決定）
　いつまでに次のアクション: （この場で決定）
[/info]
```

## サマリ通知テンプレ（2件以上のときのみ投稿）

```
[info]
[title]【月次助成金スキャン｜2026年5月｜今月は3件】[/title]
今月、HAMAYOUリゾートに関係しそうな助成金が3件見つかりました。
この後に1件ずつ詳細を投稿しますので、以下の一覧を参考に
気になるものから読んでみてください。

── すぐ判断できそうな候補 ──
③ キャリアアップ助成金｜パート→正社員登用 など
　準備にかかる時間: 余裕あり

── 今日判断したいが条件あり ──
① 中小企業省力化投資補助金｜光風閣くわるび省人化 など
　準備にかかる時間: あまりない（今週中に動く必要あり）

── もう少し情報が必要 ──
② 山梨県 観光コンテンツ造成補助
　準備にかかる時間: ふつう

この後の詳細メッセージで、それぞれの中身を確認してください。

今日この場で決めること
1. 各候補について: 採用 / 条件付き / 持越し / 見送り
2. 採用する場合の担当
3. 次のアクション期限
[/info]
```

## 準備余裕の表示ルール

リサーチ結果の「高/中/低」を以下に読み替えて表示する。補足説明がある場合はそのまま残す。

| リサーチ結果 | 表示 |
|---|---|
| 高 | 余裕あり |
| 中 | ふつう |
| 低 | あまりない |

## サマリのグループ分け

| 仮判断ラベル | サマリでの表示 |
|---|---|
| 採用候補 | すぐ判断できそうな候補 |
| 条件付き採用 / 要確認 | 今日判断したいが条件あり |
| 持越し候補 | もう少し情報が必要 |
| 見送り候補 | 参考（見送り寄り） |

---

## 運用ルール

- 投稿順: 2件以上の場合はサマリ → 個別1 → 個別2 → ...（1件なら個別のみ）
- 会議後: 個別通知への返信で **【会議決定】** を定型投稿
- 採用・条件付き採用: その場で **ChatWorkタスク化**（担当・期限）
- 投稿は **BOT専用アカウント** または `self_unread`

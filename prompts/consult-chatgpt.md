# ChatGPT Pro（助成金アドバイザー）相談テンプレ集

会議で「持越し」になった候補を、翌週の役員会議までに ChatGPT Pro に投げて判断材料を整える時に使うテンプレ集。
ChatWork の個別メッセージ末尾の「持越し時の相談テンプレ」リンクからここに飛ばす想定。

---

## 使い方

1. ChatWork の個別通知（1助成金1メッセージ）から該当制度のサマリをコピー
2. 下記テンプレのうち目的に合うものを選ぶ
3. `{{ }}` を埋めて ChatGPT Pro に貼り付け
4. 返ってきた分析を、元の ChatWork メッセージへの返信として共有
5. 翌週の役員会議で最終判断

**前提:** 機密データ（財務詳細・個人情報）はここに貼らない。会社プロファイルは `company_hamayou.md` 相当の公開可能レベルに留める。

---

## テンプレ① 適合度・メリデメ分析（標準）

持越しになった候補の「採る／採らない」を判断するための基本形。

```
あなたは中小企業の助成金アドバイザーです。
以下の助成金が、下記の会社にとって採用すべきかを分析してください。

# 会社プロファイル
- 会社名: 株式会社HAMAYOUリゾート
- 所在地: 山梨県 富士河口湖町 西湖
- 業種: 宿泊業・観光業・温泉業・キャンプ場運営
- 規模: 中小企業、正社員40〜50名（+パート・アルバイト・技能実習生）
- 主要施設: 光風閣くわるび（団体研修特化ホテル・最大500名）/ いずみの湯（日帰り温泉）/ キャンプビレッジGnome

# 助成金情報
{{ ChatWork通知の本文をここに貼る }}

# 分析してほしいこと
1. この会社との事業適合度（高・中・低 + 理由）
2. 想定される主なメリット（最大3つ）
3. 想定される主なデメリット・リスク（最大3つ）
4. 申請難易度（高・中・低 + その根拠）
5. 類似・重複する他制度があれば指摘
6. 採用すべきかのあなたの推奨（採用 / 条件付き採用 / 見送り）と理由
```

## テンプレ② 準備余裕チェック（締切が近い時）

締切までの日数が短く「間に合うかどうか」を見極めたい時。

```
以下の助成金について、下記会社が締切までに申請準備を間に合わせられるか評価してください。

# 会社プロファイル
（テンプレ①と同じ）

# 助成金情報
{{ ChatWork通知の本文 }}

# 今日の日付
{{ YYYY-MM-DD }}

# 分析してほしいこと
1. 申請までに必要な書類・証憑の一覧
2. それぞれの取得・作成に要する標準的な期間
3. 社内だけで完結するもの / 外部（商工会・社労士・ベンダー等）に依頼が必要なもの の切り分け
4. 締切までに間に合わせる場合のクリティカルパス
5. 間に合わせるために今週中にやるべきこと（最大5つ）
6. 現実的に間に合う確度（高・中・低）
```

## テンプレ③ 他制度との比較（似た制度が複数ある時）

同種の制度が複数候補にあがった時、どれを優先するかを決めるため。

```
以下の複数の助成金は似た用途のものです。この会社にとってどれを優先すべきかを比較分析してください。

# 会社プロファイル
（テンプレ①と同じ）

# 比較対象
## 候補A
{{ 本文 }}

## 候補B
{{ 本文 }}

（必要に応じて候補C以降）

# 分析してほしいこと
1. 各候補の事業適合度
2. 金額・補助率・採択率の比較
3. 併用可否（併給制限の有無）
4. 申請難易度の比較
5. 優先順位の推奨（1位→2位→3位）と理由
```

## テンプレ④ 見送り判断の裏取り

「見送り候補」になったものについて、本当に見送ってよいかを確認するため。

```
以下の助成金は社内で「見送り候補」と判断されました。
この判断が妥当かどうかをセカンドオピニオンとして評価してください。

# 会社プロファイル
（テンプレ①と同じ）

# 助成金情報
{{ 本文 }}

# 見送り理由（社内判断）
{{ 例: 事業と合わない / 規模条件で対象外 / 準備余裕がない / 既に類似制度を利用中 }}

# 分析してほしいこと
1. 社内の見送り理由は妥当か
2. 見落としている採用メリットはないか
3. 条件を変えれば採用可能になる余地はあるか（例: 対象施設を変える、申請時期をずらす）
4. 見送る場合、次回の公募に向けて今から準備しておけることはあるか
5. 最終推奨（見送り妥当 / 再検討推奨）
```

---

## 運用メモ

- ChatGPT Pro の回答は **参考意見**。最終判断は役員会議で行う
- 回答で気になった論点があれば、元の ChatWork 個別メッセージに返信として貼る
- 機密に触れる内容（具体的な財務数値、人名等）は貼らない
- 同じ制度で複数回相談した場合、最新の回答を優先
